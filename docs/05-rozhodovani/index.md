# 5. Rozhodování

<div class="lesson-meta">
<strong>Doporučený čas:</strong> 60 minut<br>

</div>
## Podmínka

```python
temperature = 21

if temperature >= 20:
    print("Je teplo.")
else:
    print("Vezmi si bundu.")
```

## Více možností

```python
score = 7

if score >= 9:
    print("Výborně")
elif score >= 5:
    print("Dobře")
else:
    print("Zkus to znovu")
```
