## Tři skupiny chyb

### Syntaktická chyba

Python nedokáže kód přečíst.

### Běhová chyba

Program se spustí, ale během práce narazí na problém.

### Logická chyba

Program doběhne, ale výsledek není správný.

## Doporučený postup

1. Přečti poslední řádek chybové zprávy.
2. Najdi typ chyby.
3. Najdi číslo řádku.
4. Zkontroluj také předchozí řádek.
5. Oprav jednu věc a znovu spusť program.
